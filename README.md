# 施工抽查流程圖生成器
# 匯出 PNG 圖片與 SVG 向量檔
這是一個以 HTML、CSS、JavaScript 製作的公共工程流程圖生成器，可用於快速建立施工流程圖，並支援匯出 PNG 圖片與 SVG 向量檔。

## 功能特色

- 新增、修改、刪除流程節點
- 支援矩形、圓角矩形、菱形、六角形節點
- 支援檢驗停留點與安衛查驗點符號標示
- 即時預覽流程圖
- 匯出 PNG 圖片
- 匯出 SVG 檔案

## 使用技術

- HTML
- CSS
- JavaScript
- SVG

## 適用情境

- 公共工程施工流程圖
- 工程品質管理文件
- 施工計畫書圖面輔助
- 工程文件電子化展示
